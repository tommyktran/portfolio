Hi! I'm Tommy Tran.

I'm a programmer specializing in web development and design. Although I am most familiar working with vanilla HTML, CSS, and JS, I have also worked with Node.js and C#.

On the design side of things, I have experience with designing and prototyping in Figma, as well as using Affinity Publisher.

I am currently working for IVS LLC, a software company which specializes in accessible voting.

My other interests include chess and creative writing. I also really like to make games from HTML/CSS/JS.

I'm interested in technology because I enjoy solving problems and coming up with solutions using logical thinking or out-of-the-box creativity. Besides that, I also enjoy learning new design systems which improve user experience.

I have graduated from Code Lousville. I have completed their Front End, Javascript, and .NET courses and am currently taking the Python course. Much of my coding and design knowledge was learned from my experience with Code Louisville, on the job at IVS LLC, or on my own, while making a personal project.

A game where you input keyboard commands to kill enemy ships that appear on the screen and try to survive as long as possible. Code Louisville Front End Project.

Blind Chess Bot

A Discord bot coded with Node.js and Discord.js that allows one to create blind chess tactics, a type of chess puzzle that improves visualization. The position before the tactic is presented, along with several moves to visualize.
